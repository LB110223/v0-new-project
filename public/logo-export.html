Smart Impulsion Logo

Instructions pour exporter le logo en PNG

  1. Faites un clic droit sur le logo ci-dessous
  2. Sélectionnez "Enregistrer l'image sous..."
  3. Choisissez l'emplacement où vous souhaitez enregistrer le fichier
  4. Sélectionnez le format PNG dans le menu déroulant des types de fichiers
  5. Cliquez sur "Enregistrer"

Vous pouvez également faire une capture d'écran du logo si la méthode ci-dessus ne fonctionne pas.
